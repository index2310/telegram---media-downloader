Telegram bot that downloads public media from TikTok and X links.

Quick start
1) npm run install:root
2) Copy .env.sample to .env and set TELEGRAM_BOT_TOKEN
3) npm run dev

Usage
Paste a public TikTok or X link into chat. The bot will fetch the media and send it back.

Commands
1) /start
2) /help

Notes
1) Public content only.
2) Telegram file limits apply.

More details are in DOCS.md.
